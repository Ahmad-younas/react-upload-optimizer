# Code of Conduct

## Our Pledge

We are committed to making participation in this project welcoming and respectful for everyone.

## Expected Behavior

- Be respectful.
- Be constructive.
- Welcome newcomers.
- Accept feedback gracefully.

## Unacceptable Behavior

- Harassment
- Personal attacks
- Hate speech
- Spam
- Discrimination

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct and may remove comments, issues, or contributions that violate these guidelines.

Thank you for helping make this community welcoming.
